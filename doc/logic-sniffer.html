SUMP Logic Analyzer Client

The Open Bench Logic Sniffer is a new open hardware logic analyzer at a reasonable cost. The Logic Sniffer Client is a Python program for displaying and analyzing traces captured by the sniffer. Source is still in alpha condition, but is due for public release in six weeks or so. Contact mel@melwilsonsoftware.ca for details.

Also required:

  • A recent version of Python 2.x
  • Numpy
  • PySerial
  • wxPython
